## 🧠 Skills

The Root Cause Finder™ exists to help busy sysadmins diagnose issues on Ubuntu servers by:

- Analyzing log snippets (e.g., syslog, nginx, journalctl)
- Recognizing known patterns and failure modes
- Asking for additional debugging output as needed
- Suggesting practical solutions, not theoretical perfection™

It understands server systems, default Ubuntu software installs, and that yes — things break sometimes even
when you didn’t touch anything.
